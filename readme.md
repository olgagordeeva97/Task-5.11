# Быстро-новости

**Быстро—новости**- это учебный проект новостного сайта, созданный с использованием **HTML** и **CSS**.

Сайт демонстрирует базовую структуру современной веб-страницы.

## Описание проекта  
Этот сайт создан в рамках учебного задания по основам вёрстки и каскадных таблиц стилей.
Проект помогает понять, как структурировать HTML-документ, применять CSS-селекторы и оформлять контент современным способом.
